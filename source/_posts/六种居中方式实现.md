```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <link rel="stylesheet" href="index.css">
</head>
<body>
    <h3>方式1：</h3>
    <p>将父元素设置为table-cell，同时设置水平和垂直居中，子元素使用内联方式即可实现垂直居中</p>
    <div class="p1">
        <div class="c1">
            内容
        </div>
    </div>
    <h3>方式2：</h3>
    <p>将父元素设置为相对定位，子元素通过绝对定位，调整margin进行垂直居中</p>
    <div class="p2">
        <div class="c2"></div>
    </div>
    <h3>方式3：</h3>
    <p>将父元素设置为flex，子元素使用margin：auto使居中</p>
    <div class="p3">
        <div class="c3"></div>
    </div>

    <h3>方式4：</h3>
    <p>将父元素设置为flex，并且设置水平和垂直的对齐属性</p>
    <div class="p4">
        <div class="c4"></div>
    </div>

    <h3>方式5：</h3>
    <p>在子元素一直宽高的情况下，将父元素设置为相对定位，子元素的上，左边距设置负一半高宽</p>
    <div class="p5">
        <div class="c5"></div>
    </div>

    <h3>方式6：</h3>
    <p>父元素设置为相对定位，子元素使用绝对定位，通过transform变换进行居中定位</p>
    <div class="p6">
        <div class="c6"></div>
    </div>
</body>
</html>
```

```css
.p1{
    height: 200px;
    width: 200px;
    border: 1px solid red;
    display: table-cell;
    vertical-align: middle;
    text-align: center;
}

.c1{
    background: blue;
    color: black;
    display: inline;
    height: 50px;
    width: 50px;
}

.p2{
    height: 200px;
    width: 200px;
    border: 1px solid red;
    position: relative;
}

.c2{
    background: blue;
    color: black;
    height: 50px;
    width: 50px;

    position: absolute;
    left: 0;
    top: 0;
    right: 0;
    bottom: 0;
    margin: auto;
}

.p3{
     height: 200px;
     width: 200px;
     border: 1px solid red;
     display: flex;
 }

.c3{
    background: blue;
    color: black;
    height: 50px;
    width: 50px;

    margin: auto;
}

.p4{
    height: 200px;
    width: 200px;
    border: 1px solid red;
    display: flex;
    justify-content: center;/*水平*/
    align-items: center;/*垂直*/
}

.c4{
    background: blue;
    color: black;
    height: 50px;
    width: 50px;
}

.p5{
    height: 200px;
    width: 200px;
    border: 1px solid red;
    position: relative;
}

.c5{
    background: blue;
    color: black;
    height: 80px;
    width: 50px;
    position: absolute;
    margin-top: -40px;
    margin-left: -25px;
    left: 50%;
    top: 50%;
}

.p6{
    height: 200px;
    width: 200px;
    border: 1px solid red;
    position: relative;
}

.c6{
    background: blue;
    color: black;
    height: 80px;
    width: 50px;
    position: absolute;

    top:50%;
    left: 50%;
    /*移动自身宽度一半的距离*/
    transform: translate(-50%,-50%);

}
```
